## Run Locally

Clone the project

```bash
  git clone https://github.com/Bang1710/perfume-web-app-mvc.git
```

Go to the project directory

```bash
  cd my-project
```

Run project option 1

```bash
  dotnet watch run
```

Run project option 2
```bash
  Select IIS Express on Visual Studio
```
